Python is a programming language. The programs that we write in the python language will be the input to the python interpreter, which is a program that runs on the computer. The Python interpreter reads our programs and executes them by following the rules of the Python language.

Lesson 4: Variables

Variables

A variable is a name that refers to a value. In Python, we can use any sequence of letters and numbers and underscores (_) we want to make a variable name, so long as it does not start with a number. For example:

Decia_stage2
Sooners
Student15

We can introduce a new variable by using an assignment statement. Once the program executes the assignment expression, the variable name will refer to the value of the expression on the right side of the assignment. We can use the variable name anywhere we want and it will mean the same thing as the value it refers to.

Test_of_speed = 435

Variables Can Vary

The value a variable refers to can change. When a variable name is used, it always refers to the last value assigned to that variable. Since the value that a variable refers to can change, the same exact expression can have different values at different times depending on when it was executed.

This gets more interesting when we use the same variable on both sides of an assignment. The right side is evaluated first, using the current value of the variable. Then the assignment is done using that value. In the following expressions, the value of hours changes from 9 to 10 and then to 20 as the expression changes:

Hours = 9
Hours = Hours + 1 #after the assignment, Hours refers to 10
Hours = Hours * 2 #after the assignment, Hours refers to 20

It is important to remember that although we use = for assignment it does not mean equality. You should think of the = sign in Python as an arrow, , showing that the value the right side evaluates to is being assigned to the variable name on the left side.

Lesson 5: Strings

Strings

A string is a sequence of characters surrounded by quotes; either single or double. The only requirement is that the string must start and end with the same kind of quote. This allows you to include quotes inside of quotes as a character in the string.

name = ‘Decia’ or “Decia”
print name
print ‘Hello’ + name

Operators on Strings

We can also use the plus sign operator (+) on strings, but it has a different meaning from when it is used on numbers. With string, plus sign means concatenation which is the joining of character strings from end-to-end. However, you cannot use the plus sign operator to combine strings and integers. If you do, you will get an error message once the program runs.

name = ‘Decia’
print name
print ‘Hello’ + name
Hello Decia

Sub-Sequences from Strings

You can select a sub-sequence of a string by designating a starting position and an end position. Python reads the characters positions starting at 0, so that if we consider the string ‘Project’ that has 8 characters, then ‘P’ would be in the 0 position. If the start expression is missing, the sub-sequence starts from the beginning of the string. If the end expression is missing, the sub-sequence goes to the end of the string.

(string){start expression:end expression}
city = ‘Atlanta’
print city[3:6]
ant

Finding Strings in Strings

The find method is a built in operation, or method, provided by Python, that operates on strings to search within a given string for a particular target expression. The output of the find is the position of the string where the specified sub-string is found. If the target string is not found anywhere in the search string, then the output will be -1.

string = ‘Stage two project is for Stage two Udacity’
print string.find(‘Stage’)
0

The find method can also pass in a number as well as a target expression. The number input is the position in the search string where the find will start looking for the target string. The output is a number giving the position of the first occurrence of the target string in the search string at or after the number input position. If there is no occurrence at or after that position, the output is -1.

string = ‘Stage two project is for Stage two Udacity’
print string.find(‘Stage’, 5)
25

Lesson 6: Control Flow and Loops

Control Flow If and While Loops

Control flow is a way to control the execution of the program statements so that it will behave as the programmer intended. One way Python programming uses this is through “If statements” and “While Loops”. If statements will evaluate the value of the first expression to see if it is “True”, and if “True” the second expression will not be evaluated. If the first expression value is evaluated to be “False”, the value will become the second expression value. On the other hand, While Loops will continue to repeat the programming statements until an evaluation of False is returned.

Example of If Statement:
if a>c:
return a
else:
return c

Example of While Loop:
i = 0
while i<10:
print i
i = i + 1

Lesson 7: Object Oriented Programming

Object Oriented Programming

Is a programming paradigm based on the concept of "objects", which are data structures that contain data, in the form of fields, often known as attributes; and code, in the form of procedures, often known as methods. In Object Oriented Programming, computer programs are designed by making them out of objects that interact with one another.There is significant diversity in object-oriented programming, but most popular languages are class-based, meaning that objects are instances of classes, which typically also determines their type.In Object Oriented programming, computer programs are designed by making them out of objects that interact with one another. There is significant diversity in object-oriented programming, but most popular languages are class-based, meaning that objects are instances of classes, which typically also determines their type.

Python's Library

The library contains built-in modules that provide access to system functionality such as file I/O that would otherwise be inaccessible to Python programmers, as well as modules written in Python that provide standardized solutions for many problems that occur in everyday programming.Some of these modules are designed to encourage and enhance the portability of Python programs.

Modules

A module is a python file that has only defenitions of variables, functions, and classes. We import bits of it (or all of it) into other programs. This assumes that the module is in the same directory as the main program with file extension (py), or is a default module that comes with Python. You normally put all import statements at the beginning of the python file. Modules are great for importing code.

Methods

Methods are just like functions, but with two main differences. Methods are defined inside a class definition in order to make the relationship between the class and the method explicit.The other one is that the syntax for invoking a method is different from the syntax for calling a function.

Classes and Objects

Classes are essentially a template or blueprint to create your objects.It is a way to take a group of functions and data and place them inside a container so you can access them with an operator. A Python class starts with the word class and is followed by the class name.

Objects are an encapsulation of variables and functions into a single entity. Objects get their variables and functions from classes.

Instance Method and Variable

The instance could be called the constructor of the class because it initializes all of the data inside of the object.This is the first piece of code executed. Instance variables need not to be declared like some variables. They exist when they are first assigned within the code. Instance variables are normally prefixed with the reserved word self. They are typically introduced and initialized in a constructor method named __init__. The instance method is a function that binds to a class object in order to return certain data about that object.

class Movie(): def __init__(self, movie_title, movie_storyline, poster_image, trailer_youtube):
self.title = movie_title
self.storyline = movie_storyline
self.poster_image_url = poster_image
self.trailer_youtube_url = trailer_youtube

Inheritance

Inheritance is the process by which one class (child class) will get most or all of its data and behavior from a "parent" class.The parent and child class can interact in three ways:

1. The actions on the child imply an action on the parent.
2. The actions on the child override the action on the parent.
3. The actions on the child alter the action on the parent.

class Parent():
def __init__(self, last_name, eye_color):
print("Parent Constructor Called")
self.last_name = last_name
self.eye_color = eye_color

class Child(Parent):
def __init__(self, last_name, eye_color, number_of_toys):
print("Child Constructor Called")
Parent.__init__(self, last_name, eye_color)
self.number_of_toys = number_of_toys
